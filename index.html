Como as Equipes Funcionam?
Crie grupos fixos de funcionários que sempre trabalharão juntos. Isso é ideal para padrões de rodízio como 12x36, 2x2, etc. Ao escalar uma equipe, todos os seus membros são alocados automaticamente.
E os Funcionários Sem Equipe?
Qualquer funcionário que não pertencer a uma equipe continuará sendo alocado individualmente pelo algoritmo para preencher as vagas restantes e complementar a cobertura dos turnos.

Backup e Restauração

Seus dados são salvos apenas neste programa. Use as opções abaixo para criar um arquivo de segurança (backup) e restaurá-lo em outro computador ou instalação do programa.

Criar um Backup Manual

Isso irá exportar todos os seus dados (turnos, funcionários, escalas, etc.) para um único arquivo .json.

Restaurar de um Backup

Atenção: A importação de um arquivo substituirá permanentemente todos os dados existentes no aplicativo.

🔥 Apagar Todos os Dados

Ação irreversível. Remove permanentemente todos os turnos, cargos, funcionários, equipes e escalas salvas deste programa.
